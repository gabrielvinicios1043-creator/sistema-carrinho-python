# 🛒 Sistema de Carrinho em Python

Projeto simples desenvolvido em Python simulando um sistema de carrinho de compras com menu interativo no terminal.

## 📌 Funcionalidades

- Adicionar produtos com nome e preço
- Listar itens do carrinho
- Calcular total da compra
- Mostrar quantidade de itens
- Encerrar programa

## 🧠 Conceitos praticados

- Estruturas de repetição (`while`)
- Estruturas condicionais (`if`, `elif`, `else`)
- Listas
- Laços `for`
- Função `sum()`
- Formatação de strings

## ▶️ Como executar

1. Baixe o arquivo `main.py`
2. Execute no terminal:

```bash
python main.py
